# Password_Manager

## Project Overview
Password_Manager is a web-based authentication demo that allows users to register and log in using securely hashed passwords.

## Project Objective
To demonstrate secure password handling and authentication mechanisms using cryptographic hashing.

## Key Features
- User registration
- Login authentication
- Password hashing using SHA-256
- No plaintext password storage

## Technologies Used
- Python
- Flask
- SQLite
- HTML, CSS, JavaScript

## Security Concepts Covered
- Password hashing
- Authentication workflows
- Secure credential storage
- Backend input validation

## Project Workflow
1. User registers with username and password.
2. Password is hashed before storage.
3. Login validates hashed credentials.
4. Authentication result is returned securely.

## Disclaimer
This project is for learning purposes and not intended for production use.
